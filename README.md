# She Can Foundation - Frontend Development Internship Task

## Overview

This project is a simple single-page website created for the She Can Foundation as part of a Frontend Development Internship Task.

The webpage showcases the NGO with a clean and user-friendly design using HTML and CSS.

## Features

* NGO Name
* Main Heading
* About Section
* Image Section
* Call-to-Action Button
* Footer
* Responsive Layout

## Technologies Used

* HTML5
* CSS3

## Project Objective

The objective of this task is to demonstrate basic frontend development skills by creating an informative and visually appealing webpage for an NGO.

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Explore the webpage.
